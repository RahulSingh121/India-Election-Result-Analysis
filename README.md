# India Election Result Analysis

## Project Overview

This project analyzes Indian election results using SQL to uncover meaningful insights from election data. Various SQL queries were used to examine voting patterns, candidate performance, party-wise results, and state-level election statistics.

## Objectives

* Analyze constituency-wise election results.
* Calculate total votes, candidates, and participating parties.
* Compare EVM votes and postal votes.
* Generate state-level election insights.
* Perform data aggregation and reporting for decision-making.

## Skills Used

* SQL
* Joins
* Aggregate Functions
* GROUP BY
* Window Functions
* Data Analysis

## Files

* **IndiaElectionResultAnalysis.sql** – Contains all SQL queries used for data exploration and analysis.
* **constituencywise_details.csv**
* **constituencywise_results.csv**
* **partywise_results.csv**
* **states.csv**
* **statewise_results.csv**

## Key Insights

* State-wise election statistics and performance metrics.
* Party-wise analysis of election results.
* Candidate-level performance evaluation.
* Comparison of EVM and postal vote distributions.
* Identification of voting trends across constituencies.

## Tools & Technologies

* SQL
* MS SQL Server

## Conclusion

This project demonstrates the use of SQL for analyzing large-scale election datasets and generating actionable insights through data aggregation, filtering, joins, and advanced querying techniques.
